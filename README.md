## What is Studio Nova?

**Studio Nova** is a premium web design agency portfolio and showcase platform built for the Indian market. It demonstrates what a modern, conversion-focused business website can look and feel like — with live, interactive demo sites for 8 different industries, a real-time project cost estimator, a fully interactive design system playground, and 13 switchable visual themes.

The platform serves as both an agency portfolio and a template showcase, letting prospective clients **experience** a finished product before committing to a project.

## The Main Website

### Hero Section
The homepage opens with a full-screen animated landing — a live **particle network canvas** that responds to your cursor, flowing ambient glowing orbs, and a cycling word animation that rolls through the types of businesses Studio Nova serves.

Four bento-grid stats anchor confidence at a glance:

| Stat | Value |
|---|---|
| Shipped Sites | 140+ |
| Lighthouse Average | 98 |
| Serviced Niches | 12 |
| Client Rating | 4.9 ★ |

### Industries — Interactive Live Templates
A bespoke tile grid showcasing all 8 industry verticals. Every tile displays a live preview image that zooms on hover, along with the visual theme used, and clicking any tile launches the full live demo for that niche.

### Process — Four Weeks from Kickoff to Live
A clean four-step bento layout that walks through the Studio Nova workflow:

1. **Discover** — Listen, audit, and map the customer journey. Workshops in week one.
2. **Design** — Two divergent directions, then sharpen. Figma prototypes you can click.
3. **Build** — Component-first development. Animations, CMS, and integrations.
4. **Launch** — QA across devices, SEO setup, analytics. Ship, then measure.

### Design System Playground
An interactive sandbox where visitors can tune a live UI card in real-time — adjusting corner radius, border width, glass blur, glow intensity, tilt physics, accent color, shimmer effects, rainbow border animations, and CTA button styles. The card responds instantly and exports ready-to-use CSS variables.

Preset styles include: **Cyberpunk · Neubrutalist · Claymorphic · Minimalist · Glassmorphic**

### Before & After Slider
A drag-to-reveal split-screen comparison showing the transformation from a basic default website to a Studio Nova redesign — demonstrating the real-world visual impact of premium web design.

### Interactive Cost Estimator
A fully functional project pricing calculator with real-time cost and timeline output:

- **Project Type** — Landing Page / Corporate Site / DTC Storefront / Custom Portal
- **Page Count** — Adjustable slider scoped to the selected project type
- **Premium Features** — Booking & Scheduling · Cart & Subscriptions · Blog & CMS · Client Portal & Auth · Calculators/Tools
- **Experience Level** — Clean UI · Kinetic Fluid · Bespoke WebGL/3D

Generates an estimated cost range and delivery timeline, then opens a pre-filled email to Studio Nova with the full configuration details.

### Client Testimonials
Three real-world quotes from clients across salon, fine dining, and real estate sectors.

### Footer
**Ahmedabad · Mumbai · Bengaluru** — with links to Instagram, Dribbble, and LinkedIn.
